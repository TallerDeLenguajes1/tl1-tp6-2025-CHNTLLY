##¿String es una tipo por valor o un tipo por referencia? 
String es un tipo por referencia, es decir que cuando una funcion utiliza una variable string, 
no se pasa una copia de la varibale sino que se pasa la direccion de memoria de la variable

##¿Qué secuencias de escape tiene el tipo string? 
// para barra invertida
/' para comilla simple
/" para comilla doble
/n para salto de linea
/0 para caracter nulo
/t para tabulacion
etc.

##¿Qué sucede cuando utiliza el carácter @ y $ antes de una cadena de texto?
@ permite ignorar las secuencias de escape
$ permite insertar variables a la cadena y/o concatenarlas
